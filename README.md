## How to Use:
clone the repository

> **#**&ensp; `git clone https://github.com/wick3dr0se/fetch.sh`

change directory into the folder

> **#**&ensp; `cd fetch.sh`

make script executable

> **#**&ensp; `chmod +x fetch.sh`

execute script

> **#**&ensp; `./fetch.sh`
