## Installation
### From package manager:
Arch Linux [AUR]: `sysfetch-git`

#### Execution:
```bash
sysfetch
```

---

### From GitHub:
clone the repository
```bash
git clone https://github.com/wick3dr0se/sysfetch
```
change directory into the folder
```bash
cd sysfetch
```

#### Execution:
```bash
bash sysfetch
```
